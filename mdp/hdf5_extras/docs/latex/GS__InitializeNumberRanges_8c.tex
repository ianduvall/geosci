G\+S\+\_\+\+Initialize\+Number\+Ranges initializes ranges for various types. 

\hyperlink{GS__InitializeNumberRanges_8c_a56293922e29b8f443d33022358bc65d0}{G\+S\+\_\+\+Initialize\+Number\+Ranges()} initializes the ranges for each of the scalar number types used in Geo\+Sci. This is needed during type conversions to avoid creating variables whose value is out-\/of-\/range.
