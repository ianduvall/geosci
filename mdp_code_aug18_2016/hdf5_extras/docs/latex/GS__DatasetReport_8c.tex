\begin{DoxyParagraph}{Details}
The report string looks like the following\+: 
\begin{DoxyCode}
[Raster] Landsat-Washita-SMEX3: Channel02; UINT8, 1024X4096
\end{DoxyCode}
 or more generally\+: 
\begin{DoxyCode}
[dataset-type] name: descriptor; datatype, dyXdx
\end{DoxyCode}
 where the first string is the type of dataset, which, besides \char`\"{}\+Raster\char`\"{}, could include\+: \char`\"{}\+Internal File\char`\"{}, \char`\"{}\+Metadata\char`\"{}, \char`\"{}\+Metadata/\+Internal-\/\+File\char`\"{}, or \char`\"{}\+Unknown\char`\"{}. The second string is the name of the Dataset, while the third is the descriptor. The next one is the datatype. See \hyperlink{hdf5mine_8h_aa4771ad431f7e00869566ecdb2b2520b}{G\+S\+\_\+\+Datatype\+As\+String()} for a list of possible datatypes. The last string is the dimensions of the raster\+: dy\+Xdx. 
\end{DoxyParagraph}
